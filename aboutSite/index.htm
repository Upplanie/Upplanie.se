O www.upplanie.se

(przetłumaczone z google) Witryna jest hostowana i wdrażana za pośrednictwem Netlify za pośrednictwem Github. Witryna została przetestowana w niektórych najnowszych przeglądarkach, w tym Chrome, Firefox, Safari i Chrome na Androida na HD i wyświetlacze podobne do HD. Jeśli używasz innej niestandardowej przeglądarki i chcesz zgłosić swój UX lub masz sugestie dotyczące ulepszenia tej witryny, napisz upplanie.webmaster@gmail.com.

The site is hosted and deployed via Netlify via Github. The site has been tested on a number of recent browsers, including Chrome, Firefox, Safari, and Chrome for Android on HD and near HD displays. If you use some other nonstandard browser and want to report on your UX, or if you have suggestions on how to improve this site, please write to upplanie.webmaster@gmail.com.
